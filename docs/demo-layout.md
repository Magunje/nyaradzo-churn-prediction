# Demo Layout Notes

These mockups are lightweight SVG stand-ins for the system screens. The implemented frontend code mirrors this structure:

- `mockups/login.svg`: branded login with secure access message and validation area
- `mockups/dashboard.svg`: KPI cards, quick actions, and recent predictions
- `mockups/records.svg`: searchable records table with row actions
- `mockups/prediction.svg`: prediction summary, risk factors, and retention action

The live React app is the primary deliverable. These mockups are included to satisfy the “mock screenshots/demo-ready layout” requirement from the prompt.
